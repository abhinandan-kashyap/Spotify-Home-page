# 🎵 Spotify Home Page Clone

## 📌 Project Overview

A responsive Spotify-inspired web interface built using **HTML5 and CSS3**.  
This project focuses on creating a clean UI, structured layouts, and improving frontend development skills.

---
## 🚀 Features

- Modern Spotify-inspired UI
- Responsive webpage layout
- Sidebar navigation design
- Organized assets management
- Clean and reusable CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Git & GitHub

---

## 📚 Key Learnings

- Web page structuring
- UI/UX implementation
- CSS positioning & layouts
- Responsive design principles
